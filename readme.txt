archivos ya minificados y listos para produccion

ademas se implementa JS 

aqui se subio el proyecto completo incluso dependencias que solo se ocupan en dev y no en produccion

carpetas para deployment:

carpeta Build
carpeta video
index.html

------------------------------------------------
gulp es para poder automatizar tareas repetitivas
Lo primero es ejecutar npm init para creear el archivo package.json

luego creo un archivo llamado gulpfile.js

la carpeta node_modules la puedo eliminar ya que ocupa mucha memoria para recuperarla se debe ejecutar en la terminal del proyect npm i

hay dos tipo de dependencias locales o de proyecto y las de desarrollo

las del proyecto son requeridas en produccion 
las de desarrollo son requeridas mientras estas desarrollando el sitio ***Para instalar un dependencia de desarrollo npm i --save-dev
